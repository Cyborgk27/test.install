# 🏦 Banking System - Unified Deployment

Este repositorio centraliza el despliegue del sistema bancario (**Banking Management**) y su interfaz de usuario (**Angular Frontend**). Utiliza Docker para garantizar un entorno consistente y scripts de automatización para facilitar la instalación.

## 📋 Requisitos Previos

Antes de comenzar, asegúrate de tener instalado:
* **Docker** y **Docker Compose V2**.
* **Git**.
* Una base de datos MySQL corriendo (opcional, el sistema puede usar una existente).

## 🚀 Instalación Rápida

Para desplegar el sistema completo sin afectar contenedores de bases de datos existentes, sigue estos pasos:

1. **Clonar este repositorio:**
   ```bash
   git clone [https://github.com/Cyborgk27/test.install.git](https://github.com/Cyborgk27/test.install.git)
   cd test.install
